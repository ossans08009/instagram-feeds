カスタマイズ可能なカルーセルまたはグリッドレイアウトでInstagramフィードを表示します。WordPress管理パネルから複数のフィードとInstagramアカウントを簡単に管理できます。

== 説明 ==

Instagram Feedsプラグインを使用すると、あなたのWordPressサイトにInstagramコンテンツを美しく表示できます。複数のInstagramアカウントを管理し、カスタムフィードを作成し、それらをカルーセルまたはグリッド形式で表示します。直感的な管理パネルにより、InstagramをWordPressサイトと簡単に統合できます。

= 主な機能 =
* カルーセルまたはグリッド形式でInstagramフィードを表示
* 複数のInstagramアカウントを管理
* カスタマイズ可能なフィード設定: ハッシュタグ、表示形式、並び順
* あらゆる画面サイズに対応するレスポンシブデザイン
* 投稿、ページ、またはウィジェットにフィードを埋め込むための簡単なショートコード

== インストール手順 ==

1. `instagram-feeds` フォルダを `/wp-content/plugins/` ディレクトリにアップロードします。
2. WordPressの「プラグイン」メニューからプラグインを有効化します。
3. WordPress管理パネルの「Instagram Feeds」メニューに移動します。
4. 「Accounts」サブメニューでInstagramアカウントを追加します。
5. 「Feeds」サブメニューでフィードを作成し、設定します。
6. ショートコード `[instagram_feed id="your-feed-id"]` を使用して、サイト上にフィードを表示します。

== よくある質問 ==

= Instagramアクセストークンはどうやって取得しますか？ =
Instagramアクセストークンを取得するには、Facebook DeveloperプラットフォームでInstagramアプリを作成し、OAuthプロセスを経る必要があります。詳細な手順はInstagram APIのドキュメントに記載されています。

= 一つのページに複数のフィードを表示できますか？ =
はい、複数のショートコードを使用して、異なるフィードIDを指定することで、一つのページに複数のフィードを表示できます。

= 対応している表示形式は何ですか？ =
現在、カルーセル形式とグリッド形式に対応しています。

== スクリーンショット ==

1. **管理パネル** - 直感的なインターフェースからフィードとアカウントを管理します。
2. **カルーセル表示** - スタイリッシュでレスポンシブなカルーセルでInstagram投稿を表示します。
3. **グリッド表示** - クリーンでレスポンシブなグリッドレイアウトでInstagram投稿を表示します。
